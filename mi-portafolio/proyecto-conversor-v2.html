Conversor SI ↔ Sistema Imperial

Programa de conversión bidireccional entre el Sistema Internacional de Unidades y el sistema anglosajón. Cubre las tres magnitudes físicas más frecuentes en Ingeniería: longitud, masa y temperatura. Primera iteración de la serie de herramientas de conversión.

C++ CLI SI Units Imperial Metrología
Ver en GitHub

Descripción Técnica

El programa implementa conversiones entre el Sistema Internacional (SI) y el sistema imperial anglosajón usando factores definidos internacionalmente. La conversión de temperatura requiere una transformación afín (no solo un factor lineal): °F = (°C × 9/5) + 32, lo que distingue este módulo del resto al no poder reducirse a una simple multiplicación.

El formato de salida usa setprecision(2) con fixed para presentar resultados limpios, adecuados para contextos de ingeniería donde la sobre-precisión puede inducir errores de interpretación.

Conversiones Implementadas

MagnitudOrigen → DestinoFórmulaPrecisión
Longitud metros → pies ft = m × 3.28084 2 decimales
Masa kilogramos → libras lb = kg × 2.20462 2 decimales
Temperatura Celsius → Fahrenheit F = (C × 1.8) + 32 2 decimales

Código Fuente

Calculadora Equivalencias V2.cpp 
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int opcion;
    double valor, resultado;

    cout << "--- Conversor SI (Metrico) a Ingles ---\n";
    cout << "1. Metros (m)    a Pies (ft)\n";
    cout << "2. Kilogramos    a Libras (lb)\n";
    cout << "3. Celsius (C)   a Fahrenheit (F)\n";
    cout << "Seleccione una opcion (1-3): ";
    cin  >> opcion;

    cout << "Ingrese el valor: ";
    cin  >> valor;

    cout << fixed << setprecision(2);

    switch (opcion) {
        case 1:
            resultado = valor * 3.28084;   // 1 m = 3.28084 ft
            cout << valor << " m = " << resultado << " ft\n";
            break;
        case 2:
            resultado = valor * 2.20462;   // 1 kg = 2.20462 lb
            cout << valor << " kg = " << resultado << " lb\n";
            break;
        case 3:
            resultado = (valor * 1.8) + 32; // Transformación afín
            cout << valor << " C = " << resultado << " F\n";
            break;
        default:
            cout << "Opcion no valida.\n";
    }
    return 0;
}
