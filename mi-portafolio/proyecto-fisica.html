Calculadora Integral de Física

Sistema de cómputo científico modular para resolver ecuaciones de Mecánica Clásica, Oscilaciones, Electromagnetismo y Termodinámica. Incluye motor de conversión de 14 factores con validación de entradas en tiempo de ejecución.

C++ Physics Thermodynamics Electromagnetism Mechanics
Ver en GitHub

Descripción Técnica

El programa implementa menús anidados en C++ donde cada módulo expone fórmulas fundamentales como funciones independientes. Se definen constantes físicas globales con alta precisión: G_EARTH = 9.81 m/s², K_COULOMB = 9×10⁹ N·m²/C² y R_GAS = 8.314 J/mol·K.

La función auxiliar toRadians() convierte grados a radianes para operaciones trigonométricas. Cada entrada es validada (división por cero, raíces de negativos, constantes de resorte positivas) antes del cómputo.

Módulos del Sistema

⚙️ Física Clásica

  • Velocidad media v = d/t
  • 2ª Ley Newton F = ma
  • MRUA: posición y velocidad
  • Energía cinética Ec = ½mv²
  • Trabajo W = Fd·cos(θ)

〜 Oscilaciones

  • Periodo T = 1/f
  • Velocidad de onda v = λf
  • Ley de Hooke F = −kx
  • Periodo resorte T = 2π√(m/k)

⚡ Electromagnetismo

  • Coulomb F = k·q₁q₂/r²
  • Campo eléctrico E = F/q
  • Ley de Ohm V = IR
  • Potencia P = VI

🌡️ Termodinámica

  • 1ª Ley ΔU = Q − W
  • Calor sensible Q = mc·ΔT
  • Gas ideal P = nRT/V
  • Eficiencia η = W/Q

Fragmento de Código

Proyecto Fisica.cpp 
// Constantes globales de alta precisión
const double PI        = 3.14159265358979323846;
const double G_EARTH   = 9.81;    // m/s²
const double K_COULOMB = 9e9;     // N·m²/C²
const double R_GAS     = 8.314;   // J/mol·K

// Conversión grados → radianes
double toRadians(double degrees) {
    return degrees * (PI / 180.0);
}

// Energía cinética: Ec = ½·m·v²
cout << "Ec = " << 0.5 * m * pow(v,2) << " J\n";

// Ley de Coulomb: F = k·q₁·q₂ / r²
cout << "F  = "
     << (K_COULOMB * q1 * q2) / pow(r,2)
     << " N\n";

// Gas ideal: P = nRT / V
cout << "P  = "
     << (n * R_GAS * T) / V
     << " Pa\n";
