Conversor de Fuerzas y Masa

Herramienta de línea de comandos para conversión precisa de unidades de fuerza y masa entre sistemas de medición. Implementa los factores de conversión del Sistema Internacional con precisión de 4 decimales usando <iomanip>.

C++ CLI Unit Conversion Mechanics <iomanip>
Ver en GitHub

Descripción Técnica

Este programa resuelve un problema concreto de la Mecánica aplicada: la necesidad de convertir unidades de fuerza y masa entre los sistemas más usados en Ingeniería. El flujo es minimal y determinista: el usuario selecciona la conversión, ingresa el valor y recibe el resultado con 4 decimales de precisión controlada por setprecision(4).

Los factores de conversión están basados en el valor estándar de la aceleración gravitacional terrestre (9.81 m/s²) y las definiciones exactas del Bureau International des Poids et Mesures.

Conversiones Disponibles

OrigenDestinoFactorBase Física
lbm (Libra-masa) N (Newton) × 0.453592 × 9.81 1 lbm = 0.4536 kg; F = mg
N (Newton) kgf (Kilogramo-fuerza) × 0.10197 1 kgf = 9.80665 N (exacto)
kg (Kilogramo) lbf (Libra-fuerza) × 2.20462 1 lb = 0.45359237 kg (BIPM)

Código Fuente

Calculadora Equivalencias V1.cpp 
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int opcion;
    double valor, resultado;

    cout << "--- Conversor de Unidades de Fuerza/Masa ---\n";
    cout << "1. Libra Masa (lbm) a Newtons (N)\n";
    cout << "2. Newtons (N)  a Kilos (kgf)\n";
    cout << "3. Kilos (kg)   a Libras Fuerza (lbf)\n";
    cout << "Seleccione una opcion (1-3): ";
    cin  >> opcion;

    cout << "Ingrese el valor a convertir: ";
    cin  >> valor;

    cout << fixed << setprecision(4);

    switch (opcion) {
        case 1:
            // 1 lbm = 0.453592 kg | F = m·g
            resultado = valor * 0.453592 * 9.81;
            cout << valor << " lbm = " << resultado << " N\n";
            break;
        case 2:
            // 1 N = 0.10197 kgf
            resultado = valor * 0.10197;
            cout << valor << " N = " << resultado << " kgf\n";
            break;
        case 3:
            // 1 kg = 2.20462 lbf
            resultado = valor * 2.20462;
            cout << valor << " kg = " << resultado << " lbf\n";
            break;
        default:
            cout << "Opcion no valida.\n";
    }
    return 0;
}
